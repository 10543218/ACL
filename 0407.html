首先請開啟檔案
D:\ACL108-2\Workbook Data Files\Workbook.acl

打開table資料夾(按table左邊的+號)

Analyze > Classify
到Classify On...的下面
擊下拉式選單
選擇Vendor_No

接下來看看右邊
看到Subtotal Fields...按鈕的下方
請使用ctrl鍵依序點擊

最後按確定

請切換到 classify 資料表
這邊開始複習如何導出新欄位

看到畫面左上方
點擊Edit > Table Layout
點底3個圖示
(Add a New Expression)
Name請填入average
接著點擊 f(x)
用輸入 或是 點選
Invoice_Amount / COUNT
讓Expression出現上面這行
點擊Ok
點擊綠色勾勾

此時沒有欄位顯示出來是正常的
請在表格空白處
按滑鼠右鍵 > Add Columns
點擊average

讓average在右邊出現
點擊ok
此時你會發現表格中多了個欄位

檢查欄位資料的正確性

這邊你會學到:
如何為資料做合理解釋

我們一樣使用 classify 資料表
請檢查:
單位成本*數量Quantity
是否等於發票金額?

題目1:如何判斷?

使用漏斗功能，輸入或點擊:
Unit_Cost * Quantity = Invoice_Amount
按Ok
你會發現這邊只出現11筆資料
這代表並不是所有資料都是
P * Q = 發票金額 !!!

題目2:為何只有11筆?

請使用漏斗功能，輸入或點擊:
Unit_Cost * Quantity <> Invoice_Amount
按Ok
這邊會出現剩下的26筆資料
我用第一筆來做講解好了

請看到攤販編號10025
它的單位成本46.51
是由5筆資料所加總而來的
數量3917也是一樣的道理
你直接這樣子做相乘
當然不會等於發票金額囉!
